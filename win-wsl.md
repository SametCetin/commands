# Windows Wsl

##### Kurulu makineleri listele :
    wsl -l

##### Çalışan makineyi dağıtıma göre kapat:
    wsl --terminate Ubuntu
    
##### Çalışan makineyi kapat :
    wsl --shutdown
    
##### Çalışan makineleri listele
    wsl --list --running

##### Debian indir ve kur 
    wsl --install -d Debian

##### Debian makinesini çalıştır
    wsl -d Debian
